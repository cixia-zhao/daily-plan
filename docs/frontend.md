# 前端页面与交互速查文档

> 最后更新：2026-06-21
> 覆盖文件：`app/templates/*.html`、`app/static/style.css`、`app/static/app.js`

## 模块职责

提供电脑和手机均可使用的服务端模板页面。浏览器只通过 JSON API 操作，不读取密钥、不自行生成规则，也不能自动确认任务。

## 文件索引

### `app/templates/base.html`

- **职责**：页面骨架、响应式 meta、顶部导航、Toast 容器和静态资源入口。
- **被依赖**：全部页面模板通过 Jinja2 `extends` 继承。

### `app/templates/index.html`

- **职责**：今日页完整流程。
- **主要区域**：日期、待审池、晨间输入、任务草稿/正式清单、执行方法、三行复盘。
- **依赖接口**：计划、任务、待审项和每日复盘 API。

### `app/templates/weekly.html`

- **职责**：完成率、完成任务、实际投入、复盘天数和静态调量建议。
- **依赖接口**：`GET /api/weekly-review`。

### `app/templates/settings.html`

- **职责**：当前阶段、AI 项目周上限、康复开关和各类别任务名称。
- **依赖接口**：`GET/PUT /api/settings`。
- **注意**：页面只能显示 AI 是否启用，不能读取或回显密钥。

### `app/static/app.js`

- **职责**：三个页面的全部浏览器交互。
- **关键函数/组件**：
  - `esc()` (L3) — 转义动态文本，防止自由文本注入 HTML。
  - `toast()` (L5) — 显示短暂状态提示。
  - `api()` (L6) — 统一 JSON 请求与错误消息提取。
  - 今日页分支 (L8) — 生成草稿、渲染计划、审批、勾选、待审处理和复盘。
  - 七日复盘分支 (L20) — 加载指标并生成不自动执行的建议。
  - 设置页分支 (L22) — 加载和保存非敏感设置。
- **依赖**：`/api/*`、模板中的固定元素 ID。
- **被依赖**：全部页面模板。

### `app/static/style.css`

- **职责**：暖色纸张视觉、任务卡、状态标签、表单、Toast 与窄屏单列布局。
- **注意**：当前为单文件压缩式 CSS，修改前先定位选择器，避免大范围格式化造成噪声。

## 前后端关系

```text
今日页 → daily-plans / tasks / carryovers / daily-reviews
复盘页 → weekly-review
设置页 → settings
```

## 已知问题与注意事项

- `app.js` 多个逻辑集中在少量超长行，后续较大改动可先规划拆分，但不要无审批重构。
- 任何新增动态内容都必须使用 `textContent` 或 `esc()`，不能直接拼入未转义 `innerHTML`。
- 最终 UI 体验由用户手动验证；不要用自动浏览器代理替代用户验收。
- 当前只做响应式页面，没有 PWA、离线缓存或前端构建链。
