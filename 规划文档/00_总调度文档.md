# 总调度文档

这个文件只用于说明四个规划文档的用途，不替代原文。

## 1. 学习规划

文件：`01_学习规划_2026.6-2027.3.md`

内容：2026.6 到 2027.3 的考研打底期规划，主线是数学、英语、C语言/数据结构，副线是 AI 小工具项目，暂缓 Linux 驱动、内核源码、完整前后端等内容。

## 2. 健身 / 康复

文件：`02_健身_康复速查版.md`

内容：右膝稳定、走路重建、臀中肌激活、膝盖稳定训练，以及前 4 周禁止跑跳冲刺的康复执行规则。

## 3. 手淫 / 睡眠

文件：`03_手淫_睡眠速查版.md`

内容：限制高刺激内容、控制夜间拖延、稳定 1 点前睡、固定起床时间，目标是先不让前一晚毁掉第二天。

## 4. 习惯 / 心流 / 数学计算

文件：`04_习惯_心流_数学计算速查版.md`

内容：图书馆学习节奏、分心回归、反刍处理、手指习惯替代、数学计算检查规则、手机隔离和低刺激休息规则。

## 使用方式

平时不要混在一个文件里改。

需要看长期方向时，看学习规划。
需要处理身体和膝盖时，看健身 / 康复。
需要处理夜晚状态时，看手淫 / 睡眠。
需要处理图书馆专注、分心和数学计算时，看习惯 / 心流 / 数学计算。
